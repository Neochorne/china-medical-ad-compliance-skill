# 医疗器械广告宣传合规要点解析

## 引言

根据您提供的项目说明及相关链接，我已对《中华人民共和国广告法》以及国家和地方（北京、上海）关于药品、医疗器械、保健食品和特殊医学用途配方食品（“三品一械”）的广告审查规定进行了学习和分析。以下将为您呈报核心内容，旨在确保您的产品宣传活动完全符合中国国内的广告审核要求，并对公域和私域流量的宣传内容进行有效区分。

核心原则是，所有与产品相关的宣传措辞**必须有明确、合法的出处**，主要依据是产品的**注册证、说明书和技术要求**，坚决杜绝任何形式的自由发挥，以防范法律风险。

## 一、 核心法律法规框架

中国的医疗器械广告活动主要受以下法律法规的约束：

1.  **《中华人民共和国广告法》**：这是所有广告活动的基础性法律，确立了广告真实、合法、不得欺骗或误导消费者的基本原则。
2.  **《药品、医疗器械、保健食品、特殊医学用途配方食品广告审查管理暂行办法》**：这是针对“三品一械”广告审查管理的专项规定，细化了广告内容的准则和禁止情形。
3.  **地方性合规指引**（如北京、上海）：这些指引在国家法规的基础上，结合地方监管实践，提供了更具操作性的合规标准和负面清单。

## 二、 医疗器械广告内容核心准则

根据上述法规，所有医疗器械广告，无论发布于何种渠道，都必须严格遵守以下准则：

- **内容一致性**：广告内容必须与药品监督管理部门批准的**产品注册证书**或许可备案的**产品说明书**内容保持一致。涉及产品名称、适用范围、作用机理等信息时，**严禁超出说明书范围**。
- **显著标明义务**：
    -   推荐给个人自用的医疗器械广告，必须显著标明“**请仔细阅读产品说明书或者在医务人员的指导下购买和使用**”。
    -   若产品注册文件或说明书中有禁忌、注意事项，广告中必须显著标明“**禁忌内容或者注意事项详见说明书**”。
    -   广告必须显著标明**广告批准文号**。
- **引证内容规范**：广告中使用数据、统计资料、调查结果、文摘、引用语等引证内容的，必须**真实、准确，并标明出处**。有适用范围和有效期限的，应明确表示。

## 三、 宣传禁止行为（负面清单）

综合各项法规，以下内容在医疗器械广告中被严格禁止。这些规定是广告创意的“高压线”，必须严格规避。

| 禁令类别 | 具体禁止内容 |
| :--- | :--- |
| **功效与安全性** | - 不得含有表示功效、安全性的**断言或者保证**（如“安全无毒”、“无副作用”）。<br>- 不得说明**治愈率或者有效率**。<br>- 不得与其他药品、医疗器械的功效和安全性进行**比较**。<br>- 不得明示或暗示成分为“天然”因而安全性有保证。 |
| **推荐与证明** | - 不得利用**广告代言人**（包括明星、网红等）作推荐、证明。<br>- 不得使用**科研单位、学术机构、行业协会、专家、学者、医师、患者**等名义或形象作推荐、证明。 |
| **诱导性与保证性内容** | - 不得含有“**热销、抢购、试用**”、“**家庭必备、免费治疗、免费赠送**”等诱导性内容。<br>- 不得含有“**无效退款、保险公司保险**”等保证性内容。<br>- 不得含有“**评比、排序、推荐、指定、选用、获奖**”等综合性评价内容。 |
| **绝对化用语** | - 不得使用“**国家级**”、“**最高级**”、“**最佳**”等用语。 |
| **疾病与治疗功能** | - 除医疗、药品、医疗器械广告外，禁止其他任何广告涉及**疾病治疗功能**。<br>- 不得使用医疗用语或易使推销商品与药品、医疗器械相混淆的用语。<br>- 不得违反科学规律，明示或暗示可以**治疗所有疾病、适应所有症状**。<br>- 不得制造“**健康焦虑**”，引起公众对健康状况不必要的担忧和恐惧。 |
| **特殊人群保护** | - 不得在针对**未成年人**的大众传播媒介上发布医疗器械广告。<br>- 不得利用不满十周岁的未成年人作为广告代言人。 |

## 四、 公域与私域宣传内容分层建议

基于上述法规，特别是对“广告”的界定，我们可以对不同渠道的宣传内容进行如下分层管理：

### 1. 公域流量 (CPD, Public Domain)

公域流量渠道（如搜索引擎付费推广、信息流广告、门户网站、大众媒体等）发布的任何内容，只要构成商业推广，即被认定为“广告”，必须严格遵守上述所有法规和审查要求。

- **核心策略**：**合规审查前置**。
    -   所有广告素材在发布前，必须通过**省级药品监督管理部门的广告审查，并取得广告批准文号**。
    -   广告内容严格限制在已批准的范围内，不得进行任何剪辑、拼接、修改。
- **内容建议**：
    -   **保守、客观**：严格依据产品说明书，客观描述产品名称、型号、规格、结构及组成、适用范围、禁忌症等。
    -   **提示清晰**：按规定显著标明所有必要的提示语和警告信息。
    -   **避免擦边球**：完全避免使用任何可能产生误导的词汇、图片或暗示。
    -   **仅宣传产品名称**：根据法规，仅宣传产品名称的广告无需内容审查，但在宣传时仍需标注医疗器械注册证号。这可以作为一种合规的品牌曝光策略。

### 2. 私域流量 (Private Domain)

私域流量（如品牌自有的App、微信公众号、用户社群、一对一沟通等）的管理相对复杂。虽然看似更封闭，但一旦内容具有营销推广性质，仍可能被认定为“广告”或“变相广告”，从而受到监管。

- **核心策略**：**内容性质区分**。
    -   **非广告内容**：可以发布不以直接推销产品为目的的**健康科普知识**、**疾病风险教育**、**用户故事分享**（需注意保护隐私且不得构成推荐证明）等。上海市的指引明确，介绍健康、养生知识，如果在同一页面或同时不出现产品购物链接、销售者联系方式等，原则上不被视为广告。
    -   **广告内容**：如果要在私域发布直接的产品宣传，依然需要遵循广告法规定。例如，通过公众号文章详细介绍产品功能，若超出了说明书范围或含有功效保证，同样违法。
- **内容建议与风险提示**：

| 内容类型 | 建议操作 | 风险与注意事项 |
| :--- | :--- | :--- |
| **健康科普/知识教育** | - 围绕与产品相关的疾病领域，提供科学、客观的知识。<br>- 介绍行业技术进展、研究成果（需注明出处）。 | - **不得与具体产品直接挂钩**。例如，可以科普光疗技术，但不能在同一篇文章中引导购买特定光疗产品。<br>- 避免以科普之名，行广告之实。 |
| **用户操作指南/售后服务** | - 制作详细的产品使用教程、保养说明、常见问题解答。 | - 内容应为纯粹的指导性信息，不得夹带未经审批的宣传语。 |
| **临床数据与文献** | - **面向专业人士**（如在设有访问权限的医生专区），可以分享产品的临床试验数据、学术论文等专业资料。 | - **必须明确受众**。处方药和部分专业医疗器械的广告只能在指定的医学、药学专业刊物上发布。此原则可类推至私域的专业渠道管理。<br>- 确保数据真实、准确，并完整呈现，不得断章取义。 |
| **直播与用户互动** | - 上海市指引明确提出“**不宜以网络直播形式营销‘三品一械’产品**”。 | - 风险极高。直播中即兴的口头宣传很难完全控制在合规范围内，极易出现违禁词汇或保证性承诺，建议**绝对避免**通过直播带货。 |

## 五、 总结与行动建议

1.  **建立内部审核流程**：所有对外发布的宣传物料，无论渠道，都应建立严格的内部法务和合规审核流程，对照法规清单逐一检查。
2.  **以说明书为唯一蓝本**：市场和创意团队在构思宣传文案时，必须以产品说明书为唯一的创作依据，将合规内化为创意的起点。
3.  **区分受众与渠道**：为“公域/大众”和“私域/专业人士”准备两套不同尺度和口径的沟通材料。公域求合规，私域（专业渠道）求深度。
4.  **动态关注法规变化**：广告法规和地方监管要求在不断更新，需保持对国家及地方市场监督管理部门、药品监督管理部门官方信息的持续关注。

通过严格遵守上述规则，您的产品宣传将能最大程度地规避法律风险，确保在国内市场的稳健运营。
