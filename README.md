# China Medical Device Advertising Compliance Skill

A **Manus Agent Skill** that provides comprehensive guidance on China's laws and regulations for medical device advertising. Use this skill to review advertising copy, ensure compliance with national and local regulations, and differentiate between rules for public and private promotional channels.

## What This Skill Does

- Reviews advertising copy for compliance with Chinese medical device advertising laws
- Identifies prohibited content (absolute terms, efficacy guarantees, expert endorsements, etc.)
- Checks for required disclaimers and mandatory statements
- Provides channel-specific advice for **public domain (CPD)** vs. **private domain** marketing
- Suggests compliant alternatives for non-compliant copy

## Key Regulations Covered

- 《中华人民共和国广告法》(Advertising Law of the PRC)
- 《药品、医疗器械、保健食品、特殊医学用途配方食品广告审查管理暂行办法》
- 《医疗器械广告审查办法》
- Local guidelines from Beijing and Shanghai

## How to Install

1. Download or clone this repository
2. Place the `china-medical-ad-compliance` directory in your Manus skills folder (`/home/ubuntu/skills/`)
3. The skill will be automatically available to Manus

## Core Principle

All advertising activities for medical devices in China must be **truthful, legal, and based on official documentation**. The primary sources for any claims must be the product's **registration certificate, approved manual, and technical specifications**.
